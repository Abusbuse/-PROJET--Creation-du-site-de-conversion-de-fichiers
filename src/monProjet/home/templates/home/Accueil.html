Bienvenue sur le site de conversion de fichier

Ce site représente un projet encadrant un stage de 2ème année du BUT Informatique à l'Université de Lorraine pendant 2 mois.

Le but de ce site est de permettre à l'utilisateur de convertir un fichier de plusieurs types en unte autre extension.

Vous pouvez accéder à la page répertoriant les différents types de fichiers que vous pouvez convertir en cliquant sur le lien ci-dessous.

→ CONVERTIR

Le site est actuellement en version beta, il est donc possible que certaines fonctionnalités ne soient pas encore disponibles.
